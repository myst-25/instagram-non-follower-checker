# Instagram Non-Follower Checker

> A Tampermonkey userscript that **auto-collects** your Instagram Followers & Following lists and shows exactly who doesn't follow you back.

## ⚡ Install

> Make sure you have [Tampermonkey](https://www.tampermonkey.net/) installed first.

[![Install Script](https://img.shields.io/badge/Install-Tampermonkey%20Script-brightgreen?style=for-the-badge&logo=tampermonkey)](https://github.com/myst-25/instagram-non-follower-checker/releases/latest/download/instagram-non-follower-checker.user.js)

Or install the latest development version directly from `main`:

[Install from main branch (raw)](https://github.com/myst-25/instagram-non-follower-checker/raw/main/instagram-non-follower-checker.user.js)

---

## 🚀 How to Use

1. Go to [instagram.com](https://www.instagram.com) and open **your own profile**
2. A floating panel will appear at the bottom-right corner
3. Click your **Following** count to open the popup → press **Collect Following**
4. Wait for collection to finish (auto-scrolls the list for large accounts)
5. Close and open **Followers** popup → press **Collect Followers**
6. Press **Compare** — results show who doesn't follow back
7. Use **Copy Results** or **Download TXT** to export

---

## 🖼️ Features

- ✅ **Auto-scroll & collect** — handles accounts with thousands of followers/following
- ✅ **Stop at any time** — partial results are saved automatically
- ✅ **Compare** following vs followers in one click
- ✅ **Export** results as clipboard copy or `.txt` download
- ✅ **Live status counter** while collecting
- ✅ **Hide/show panel** toggle
- ✅ No external servers — everything runs locally in your browser

---

## ⚠️ Notes

- Works only on **your own profile** on the **desktop web version** of Instagram
- Instagram lazy-loads lists, so large accounts take longer — let the script finish
- Use responsibly. Instagram's Terms of Service prohibit unauthorized automated collection
- This script only reads what is already displayed in the page popup — it does not call any private API
